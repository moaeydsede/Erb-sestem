# ERP محاسبة + تصنيع + مراكز كلفة (Frontend فقط)

نسخة تعمل على GitHub Pages بدون سيرفر (LocalStorage).

## تسجيل الدخول
- admin / admin123

## الميزات
- قيود يومية Double-Entry مع مركز كلفة لكل سطر
- مراكز كلفة (Cost Centers)
- أصناف: مواد خام RM / إنتاج تام FG / خدمات SRV
- شراء مواد خام مع تحديث متوسط التكلفة (Moving Average)
- تصنيع:
  - BOM (قائمة مواد)
  - أوامر إنتاج (WO)
  - صرف مواد إلى WIP
  - تحميل تكاليف تشغيل/أجور إلى WIP
  - تسييل إلى FG وإغلاق الأمر
- بيع إنتاج تام مع قيد تكلفة مبيعات (COGS)
- سندات قبض/صرف
- تقارير: ميزان مراجعة، أرباح وخسائر، تقرير مركز كلفة، تقرير تكلفة أوامر الإنتاج
- Backup/Restore JSON + تصدير Excel
